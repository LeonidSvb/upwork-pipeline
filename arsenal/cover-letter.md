# Cover Letter Formula

## Structure
1. Hook (Unicode bold) — конкретный результат релевантный джобу
2. 2-3 кейса — ниша + число + срок
3. Loom — "I recorded a 3-min Loom showing [конкретно что]"
4. 1-2 умных вопроса по джобу
5. Speed line — "I already have [X] built, so this takes days not weeks"
6. Soft CTA — "Happy to jump on a quick call or send more details"
7. Если в джобе написано "start with X" — всегда делать

## Hook Formulas
- 𝗜'𝘃𝗲 𝗯𝘂𝗶𝗹𝘁 𝗲𝘅𝗮𝗰𝘁𝗹𝘆 𝘁𝗵𝗶𝘀 — [result] for [niche] in [timeframe]
- 𝗦𝗶𝗺𝗶𝗹𝗮𝗿 𝗽𝗿𝗼𝗷𝗲𝗰𝘁: [metric] in [timeframe] — here's how
- 𝗬𝗼𝘂𝗿 𝗽𝗿𝗼𝗯𝗹𝗲𝗺: [their pain]. 𝗠𝘆 𝘀𝗼𝗹𝘂𝘁𝗶𝗼𝗻: [what you do]

## Loom Script Template
1. Открыть — [релевантный workflow/демку/кейс]
2. Сказать — "I saw your job, here's what I'd build / already built"
3. Показать — работающий процесс, не слайды
4. Закончить — "This is exactly what I'd do for you, happy to discuss"
Длина: 2-4 минуты максимум

## По типу джоба
- Cold outreach / lead gen → кейс + Loom с working sequence
- Workflow build (n8n/Make) → показать похожий workflow живьём
- Airtable → показать базу структуру живьём
- HTML демка → сделать кастомную под клиента за 10 мин
- Tutoring → показать что понимаешь глубже чем клиент
- Small fix → короткий cover, без Loom, быстрый старт
- Новый клиент без истории → упор на демку, меньше текста
