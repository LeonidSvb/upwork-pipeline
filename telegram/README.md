# Telegram

Бот для получения джобов и управления пайплайном.

## Файлы

**bot.js** — главный файл, запускается через `start-bot.bat`.
Обрабатывает все команды и кнопки обратной связи.

**callbacks.js** — старый обработчик (частично дублирует bot.js, не удалён на всякий случай).

**ideas.js** — обработка сообщений из топика IDEAS → LLM → Airtable.

## Команды

| Команда | Что делает |
|---|---|
| `/start_jobs` | Включить мониторинг каждые 20 мин (age: 30 мин) |
| `/stop_jobs` | Выключить мониторинг |
| `/run` | Разовый скрейп за последние 2 часа |
| `/status` | Статус мониторинга, время последнего запуска |
| `/help` | Список команд |

## Кнопки на джобах

- **Откликнулся** — сохраняет `applied`, убирает кнопки
- **Интересно** — сохраняет `maybe`, меняет кнопки на "Откликнулся / Скрыть"
- **Не моё** — просит написать причину текстом, затем удаляет сообщение
